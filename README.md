# PhoneUAV-server
Allows user to control RC plane through Wi-Fi. You need [USC-16 servo controller](https://www.google.com/search?q=USC-16+servo+controller), Android Lollipop enabled smartphone, laptop, and analog gamepad. You can download the most recent build here: [PhoneUAV-debug.apk](https://github.com/mareksuma1985/PhoneUAV-server/blob/master/PhoneUAV/build/outputs/apk/debug/PhoneUAV-debug.apk). For MAVLink support see also: [MAVLink UDP Android Example](https://github.com/mareksuma1985/mavlink/tree/master/examples/android).
