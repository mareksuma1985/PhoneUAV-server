# PhoneUAV-server
Enables user to control RC plane through Wi-Fi. You need [USC-16 servo controller](https://www.google.com/search?q=USC-16+servo+controller), Android Lollipop enabled smartphone, laptop, and analog gamepad. For MAVLink support, see: [MAVLink UDP Android Example](https://github.com/mareksuma1985/MAVLink)
